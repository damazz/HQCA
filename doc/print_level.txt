Contains information for printing and options. Currently there are several print
options:
pr_g - print_global
    controls global printing settings

pr_m - print_main/print_method
    controls printing for a particular method

pr_o - print_optimizer
    controls printing for an optimizer

pr_q - print_quantum 
    controls printing for the quantum computer

pr_g:
=0: no output
=1: headings, macro algorithm convergence

pr_m:
=0: no output
=1: no output
=2: diagnostic output

pr_o:
=0: no output
=1: iteration output
=2: optimizer steps

pr_q:
=0: no output
=1: no output
=2: starting diagnostics #####
